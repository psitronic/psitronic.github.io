About me

I'm a PhD in theoretical physics and self-taught software developer. While I'm proficient in implementing complex algorithms for numerical computations and data analysis, my real expertise is in designing and building real-time data acquisition systems.

My personal interests also include back-end web development, in particular, API microservices and their use for building IoTs. I'm passionate about algorithms, AI and blockchain technologies.

Highlights:

  • Built a data acquisition system in Python to control complex scientific equipment/devices and monitor/acquire various parameters (temperature, pressure, etc.) via different interfaces (GPIB, LAN, serial, etc.). This software allows to operate automatically the Dilution Refrigirator with a Nuclear Demagnetization Stage (~€ 1'000'000) at Vienna Technical University, Austria, saving hundreds personnel hours.
  • Created libraries for numerical computing in Fortran (matrix diaganolization, optimization procedures, self-consistent solution of equations, etc.) which allowed to calculate physical properties of different materials and explain scientific experimental data.
